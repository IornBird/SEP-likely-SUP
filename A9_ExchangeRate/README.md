# A9_ExchangeRate - 匯率轉換器

**此文件由AI撰寫，個人強烈建議用Eclipse IDE建置**

## 📋 專案說明

A9_ExchangeRate 是一個即時匯率轉換的JavaFX應用程式，支援美元、台幣、日圓、歐元和人民幣等五種主要貨幣的兌換計算。操作簡單直觀，是學習GUI開發與金融計算的絕佳範例。

## ✨ 功能特性

- **匯率轉換**：在5種貨幣間進行轉換計算 (注意匯率是寫死在程式碼中的，不要真的參考)
- **支援貨幣**：美元、台幣、日圓、歐元、人民幣
- **貨幣交換**：一鍵互換轉換源與目標貨幣
- **即時計算**：輸入金額後立即顯示轉換結果
- **使用者友善**：簡潔清晰的UI介面設計

## 🛠️ 技術棧

## 💡 使用說明

1. **選擇來源貨幣**：從左側下拉清單選擇要轉換的貨幣
2. **選擇目標貨幣**：從右側下拉清單選擇要轉換成的貨幣
3. **輸入金額**：在上方文字欄位輸入要轉換的金額
4. **按下轉換按鈕**：計算並顯示轉換結果
5. **互換貨幣**：按下「↔」按鈕可快速交換來源與目標貨幣

## 🔧 核心邏輯

### Cal_main.java

- 主應用程式入口，繼承自 `Application`
- 負責 JavaFX UI 元件的建立與佈局
- 包含 TextField（金額輸入）、ComboBox（貨幣選擇）、Button（轉換/交換）

### Action.java

- 實現轉換計算邏輯
- `convert()` 方法：執行匯率轉換計算
- `exchange()` 方法：交換來源與目標貨幣
- `rate[]` 陣列：儲存各貨幣相對於美元的匯率

## 📋 依賴項

- **javafx-controls**: JavaFX UI 控制項
- **javafx-fxml**: FXML 標記語言支持
- **junit-jupiter-api/engine**: JUnit 5 測試框架

## 🎯 轉換公式

```
目標金額 = 輸入金額 × (目標貨幣匯率 ÷ 來源貨幣匯率)
```
